Implemented virtual function, called by dispatcher when an event has fired.

Parameters

CgiHandler is registered twice on Dispatcher if the CGI request was a POST method. Two pipes are created, one for writing to the CGI process, one for reading from it. So first we check which pipe FD is ready with an event, then read/write as necessary
